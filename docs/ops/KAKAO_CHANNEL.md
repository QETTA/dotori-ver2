## 1. 개설 정보

**채널명:** 도토리 / **검색ID:** @dotori
**사업자:** 주식회사 레인피플 (160111-0700058)
**카테고리:** 서비스 > 정보/데이터 (또는 생활 > 육아/출산)

### 개설 준비물
- 레인피플 사업자등록증 (jpg/png/pdf, 10MB 이하)
- 법인 대표자 휴대폰 본인인증 (남편 명의)
- 도토리 로고 640x640px (프로필), 800x400px (웰컴 이미지)

### 개설 절차 (center-pf.kakao.com)
1. 카카오비즈니스 통합가입 (카카오계정 로그인)
2. [+시작하기] → "아니오, 매장이 없어요"
3. 사업자등록번호 입력
4. 신청인: "법인사업자 대표" (남편 직접) 또는 "직원" (본인)
5. 채널 정보 입력 (이름, 카테고리, 검색ID, 소개)
6. 추가서류 — 사업자명≠채널명 소명:
   "주식회사 레인피플은 IT 법인으로, 도토리는 당사가 운영하는 육아 시설 정보 서비스 브랜드명입니다."
7. 비즈니스 채널 심사 (1~3 영업일)

## 7. 비즈월렛 & 소상공인 캐시

### 신청 절차
1. 파트너센터 > 비즈월렛 생성
2. 카카오 프로젝트 단골 채널 → [신청하기]
3. 최대 1개월 내 30만원 무상캐시 충전
4. 유효기간: 1년 / 현금 인출 불가

### 발송 단가
- 일반 메시지: 15원/건 (VAT별도)
- 타겟 메시지: 20원/건 (VAT별도)
- 30만원 ÷ 15원 = 약 20,000건 발송 가능

템플릿 코드: DOTORI_INTEREST_001
```

📌 현황: #{상태}

```
버튼: [도토리에서 확인하기] → 시설 상세

### 주간 TO 브리핑 (와이드 이미지형, 타겟)
- 대상: 지역+자녀연령별 친구 그룹
- 주기: 매주 월요일
- 내용: 해당 지역 신규 TO 시설 요약

### 시설 소개 (B2B 영업용)
- 대상: 시설 관계자
- 내용: 프리미엄 서비스 소개 + 무료 체험 안내

## 9. 사업자 변경 절차 (예비창업패키지 선정 후)

1. 새 사업자등록증 발급
2. 카카오 비즈니스 관리자센터 > 서비스관리 > 사업자 정보
3. 내 사업자 정보 관리 > 수정 요청
4. 사업자등록번호/대표자명 변경 + 서류 첨부
5. 심사 후 반영 (채널명/친구수/이력 유지)
6. 새 사업자번호로 소상공인 캐시 2차 신청 (30만원 추가)
