### 1-2. 출생아 수 반등 시그널

- 2025년 출생아 수: **약 258,242명** (2021년 이후 4년 만 최대)
- 합계출산율: 0.75(2024) → **0.8(2025 전망)**
- 2025년 5월까지 전년 동기 대비 **+6.4%** (16개월 연속 증가)
- **보육시설 입소 시점**: 2027-2028년 → 시설 간 경쟁이 더 심해진 상태에서 수요 회복
- 출처: [한국경제](https://www.hankyung.com/article/2025122439321), [통계청](https://www.kostat.go.kr/)

### 1-3. 유보통합 (2026 시행)

| 항목 | 변경 내용 |
|------|----------|
| 기관 통합 | 유치원+어린이집 → '영유아학교(가칭)' |
| 돌봄 시간 | 기본 8시간 + 아침/저녁 4시간 = **12시간** |
| 교사 비율 | 0세 1:3 → **1:2** |
| 교육과정 | 0~5세 통합과정 (2027년까지) |
| 시범 운영 | 경기도 40개소 거점/연계형 돌봄 |

- 부모의 검색 기준 자체가 바뀜 → **통합 검색 플랫폼 수요 급증**
- 출처: [한국경제](https://www.hankyung.com/article/2024062761915), [유보통합 정리](https://hanipinfo.com/)

### 1-4. 보육료 지원 확대

- 2025년 7월: 0~2세 보육료 단가 **5% 인상**
- 5세 무상보육 확대: 2025 하반기 5세 → 2026년 4~5세 → 2027년 3~5세
- 출처: [정책브리핑](https://www.korea.kr/briefing/pressReleaseView.do?newsId=156723261)

### 1-5. 시장 기회 종합

```
공급 과잉(시설 감소 느림) + 수요 반등(출생아 증가)
= 살아남는 시설의 "선택받기 위한 마케팅" 절실
= 도토리 B2B 솔루션 수요
```

**시즌별 기회 매핑:**

| 시기 | 이동 수요 트리거 | 규모 | 도토리 접점 |
|------|-----------------|------|------------|
| 2월 | 설명회 후 실망, 반편성 결과 공지 | ★★★ | "이동 고민 중이라면" 맘카페 바이럴 |
| 3월 초 | 새 학기 반편성 최종 확정 후 실망 | ★★★ | 반편성 비교 기능 CTA |
| 3~4월 | 새 학기 적응 실패, 교사 첫인상 불만 | ★★ | 빈자리 알림 구독 유도 |
| 상시 | 교사 교체, 아동학대 뉴스, 국공립 당첨 | ★★ | 카카오 채널 상시 운영 |
| 8월 | 2학기 반편성, 이사 시즌 | ★★ | 지역 기반 탐색 |

**서울 폐원 위기 786개소(2026)** → 폐원 예고 어린이집 이용 부모 이동 수요 급증 → 도토리 긴급 알림 기능 기회

## 3. 타겟 고객 & Pain Points

### 3-1. B2C: 이동 수요 부모 (초기 타겟)

> "지금 다니는 어린이집, 정말 괜찮으신가요?"

**이동 수요 Pain Points 상세** (보건복지부 만족도 조사 2022-2023, 육아정책연구소 기반):

**1순위: 반편성 불만** (이동 트리거 1위, 3월 피크)
- "형제끼리 같은 반 해달라는데 규정상 안 된다고 함"
- "선생님 좋아했는데 새 학기 다른 반 배정"
- 연령 기준 원칙 편성 → 형제분리, 혼합반 허가 까다로움

**2순위: 교사 잦은 교체 / 원장 불신** (연중)
- "선생님이 한 달 만에 또 바뀌었어, 이제 몇 번째야"
- 보육교사 이직률 높음 (처우 열악), 자격 취소 77% 아동학대 관련

**3순위: 설명회 후 실망** (2월 집중)
- "설명회 분위기가 너무 딱딱함, 원장이 규정만 읊더라"
- "사진으로 보던 것보다 시설이 너무 낡았어"

**4순위: 국공립 대기번호 당첨 vs 현재 민간** (연중 랜덤)
- "대기 200번이었는데 갑자기 연락 옴, 어떻게 해"
- 국공립 이용 비율 21.4% — 대기→당첨 시 비교 니즈 급증

**5순위: 유보통합 혼란** (2026 신규)
- "2026년부터 바뀐다는데 우리 애 다니는 곳은 해당이 돼?"
- 기관별 차등 적용 → 정보 니즈 발생

**6순위: 아동학대 불안** (상시, 사건 뉴스 시 폭증)

**7순위: 탐색 비용 과다** → "옮기고 싶은데 전화 돌리다 지쳐서 그냥 다님" = **탐색 자동화가 킬러 피처**

| 이동 트리거 | 시기 | 강도 |
|------------|------|------|
| 설명회 후 실망 | 2월 | ★★★ |
| 반편성 확정 후 불만 | 3월 초 | ★★★ |
| 교사 교체/불신 | 연중 | ★★ |
| 국공립 대기번호 당첨 | 연중 랜덤 | ★★ |
| 유보통합 혼란 | 2026 3월~ | ★★ |
| 아동학대 뉴스 | 연중 | ★ |
| 탐색 비용 과다 | 연중 | ★ |

### 3-2. B2B: 정원 미달 어린이집

| Pain Point | 현재 해결 방식 | 비용 |
|-----------|--------------|------|
| 원아모집 | 네이버 블로그 대행 | 월 10-50만 원 |
| 시설 홍보 | 맘카페 체험단 | 월 수만~수십만 원 |
| 키즈노트 광고 | 타겟 푸시 (건당 80-87원) | CPM |
| 오프라인 | 현수막/전단 | 건당 수만 원 |

**핵심 인사이트**: 원장들은 이미 원아모집에 월 10-50만 원 지출 중. 도토리가 더 효율적인 채널을 제공하면 즉시 전환 가능.

- 보육 분야 전용 B2B SaaS는 한국에서 **거의 전무** (블루오션)

## 10. v3.0 시장 확장

### 유보통합 기회 (2026 시행)

유치원+어린이집 통합 = 부모 검색 기준 변화 → **통합 검색 플랫폼 수요 급증**
- 기존 TAM 27,387(어린이집) → **35,887** (+ 유치원 8,500)
- 유보통합으로 시설 유형 구분 사라짐 → 도토리의 통합 DB가 선점 우위

### B2B2B 수익 모델

- **AI바우처 공급사 화이트라벨**: 연 500~3,000만원/공급사
- 보육 분야 전용 B2B SaaS는 한국에서 거의 전무 (블루오션)
- 한그루ERP(입소 후 운영)와 도토리(입소 전 매칭)는 **보완 관계 → 파트너십** 가능

### B2G 지자체 라이선싱

- CoDMON(일본) 모델 참조: 지자체에 보육 매칭 시스템 라이선싱
- 연 1~3억/지자체, 전국 226개 기초자치단체 대상
- 서울 786개소 폐원 위기 → 지자체의 시설 이동 관리 수요 발생

### 전자서명 프리미엄

- 7종 서류 전자서명 = 입소 프로세스 2~3시간 → **10분**
- CPA 건당 과금에 전자서명 포함 시 **+50% 프리미엄** (5,000원 → 7,500~15,000원)
- 감사추적인증서 = 어린이집 평가제 증빙 자료로 활용 가능
