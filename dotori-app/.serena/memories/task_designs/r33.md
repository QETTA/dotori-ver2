# R33: 2026 Organic Refinement — "숲속의 도토리"

## Design Direction
- **Tone**: Organic/Natural + Refined Minimalism — 따뜻한 자연 소재, 깨끗한 선
- **Differentiation**: Earthy depth — 숲 캐노피를 통해 보는 듯한 레이어드 서피스
- **2026 Trend Keywords**: Cloud Dancer, Bento Grid, 빨리빨리 정보 밀도, Status Pulse, Purposeful Motion

## CONTEXT
홈 화면 상태 카드 숫자가 본문과 동일 크기(15px)로 시각 계층 부재.
전체 페이지 동일한 시각 가중치로 핵심 지표 인지가 어려움.
빈자리 있는 시설의 긴급성 시각 신호 없음.

## GOAL
홈 화면 Bento Grid + 메트릭 강조 + 상태 펄스로 2026 트렌드 정합 기반 구축

## NON-GOALS
- Explore 페이지 구조 변경 (Phase 2)
- Chat 페이지 개선 (Phase 2)
- 새 컴포넌트 생성
- Catalyst 내부 수정

## AC
1. 홈 상태 카드가 Bento Grid (2열, 첫 카드 전체 너비) 레이아웃
2. 주요 메트릭 숫자가 h1(24px) 이상으로 강조
3. 빈자리 > 0일 때 상태 카드에 forest 계열 펄스 글로우
4. Cloud Dancer 톤 배경 그래디언트 적용
5. FacilityCard 상태 도트에 available 펄스 추가
6. 빌드 0 에러, 테스트 165개 통과
7. 다크모드 호환 유지

## SCOPE FILES (6개)
1. `src/app/globals.css` — Cloud Dancer bg + status-pulse keyframe
2. `src/lib/design-system/tokens.ts` — Bento 토큰 + 메트릭 강조 토큰
3. `src/lib/motion.ts` — bento stagger variant
4. `src/app/(app)/_components/HomeHero.tsx` — Bento Grid + 대형 수치
5. `src/app/(app)/_components/HomeBriefing.tsx` — 메트릭 강조
6. `src/components/dotori/FacilityCard.tsx` — 상태 펄스
